# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. By participating in this project, you agree to abide by its terms.

## Enforcement

Instances of unacceptable behavior may be reported by opening a private conversation with the project maintainer through [GitHub](https://github.com/niclaslindstedt/zag). All reports will be reviewed and handled confidentially.
