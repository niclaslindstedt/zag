# zag man

Show manual pages for agent commands.

## Synopsis

    zag man [command]

## Description

Displays detailed documentation for agent commands. When called without arguments, shows the general agent manual page. When called with a command name, shows the manual for that specific command.

## Arguments

    command    Command to show help for: run, exec, review, config, man

## Examples

    zag man             General agent overview
    zag man exec        Manual for the exec command
    zag man config      Manual for the config command
