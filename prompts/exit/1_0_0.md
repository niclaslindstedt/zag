## How to end this session

This session was launched with `--exit`. When you have produced the result, terminate the session by running one of:

```
zag ps kill self "<result>"
```

or, if your result is in a file:

```
zag ps kill self --file <path>
```

`zag ps kill self` is how the session ends. Do **not** call it until you have produced the result. Do **not** just print the result and stop — the session keeps running until kill is called.

{HINT_SECTION}{JSON_INSTRUCTION}{SCHEMA_INSTRUCTION}
If `zag ps kill self` exits non-zero, read the stderr message: it tells you what was wrong with your result (empty, invalid JSON, failed schema validation). Fix the result and call kill again.
