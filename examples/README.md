# Examples

Complete projects demonstrating [zag](https://github.com/niclaslindstedt/zag) usage patterns.

| Example | Description | Key features |
|---------|-------------|-------------|
| [cv-review](cv-review/) | Two-pass CV review pipeline | `zag-lib` API, JSON schema validation, parallel agents |
| [orchestration](orchestration/) | Shell scripts for multi-agent patterns | `spawn`, `wait`, `pipe`, `collect`, fan-out, DAG workflows |
| [react-claude-interface](react-claude-interface/) | React web chat interface | `zag exec`, `zag input`, SSE streaming, NDJSON events |
